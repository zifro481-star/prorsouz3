# Мой Профсоюз — Часть 1: Авторизация, навигация, дашборд, профиль

## Что будет создано

Мобильное приложение «Мой Профсоюз» — клиент профсоюзной платформы с подключением к API profsouz.info.

---

### **Дизайн**
- Тёмная тема по умолчанию: фон #09090b, карточки #18181b
- Основной акцент — синий #3b82f6
- Стиль: premium minimal, чистый и современный
- Шрифт заголовков: Montserrat ExtraBold (Google Fonts)
- Мягкие тени, скруглённые углы, плавные анимации

---

### **Экраны**

**1. Экран входа**
- Логотип и название приложения сверху
- Поля ввода Email и Пароль
- Кнопка «Войти» с анимацией загрузки
- Ссылка на экран регистрации

**2. Экран регистрации**
- Поля: ФИО, Email, Телефон, Пароль
- Выбор профсоюза из списка (загружается с сервера)
- Выбор подразделения (зависит от профсоюза)
- Чекбоксы согласий (оферта, обработка ПД)
- Кнопка «Зарегистрироваться»

**3. Главная (Дашборд)**
- Заголовок «Мой Профсоюз» с колокольчиком уведомлений и бейджем
- 3 быстрых действия: Обращение, Юристу, Чаты
- Баннер «Чат с ИИ — мгновенная консультация»
- 3 метрики: Активные обращения, Непрочитанные, Решено
- Блок «Первые шаги» — чек-лист с прогресс-баром
- Блок «Новости» — горизонтальный скролл карточек

**4. Уведомления**
- Список уведомлений с временем
- Непрочитанные выделены фоном
- Отметка как прочитанных

**5. Чаты (заглушка)**
- Пустой экран-заглушка для будущей реализации

**6. ИИ-чат (заглушка)**
- Пустой экран-заглушка, центральная кнопка таба

**7. Профсоюз (заглушка)**
- Пустой экран-заглушка для будущей реализации

**8. Профиль**
- Аватар — круг с инициалами на синем фоне
- ФИО, email, номер участника (MP-XXXXXX)
- Бейджи роли и статуса
- Информация: профсоюз, подразделение
- Редактируемые поля: ФИО, телефон, город, профессия
- Кнопка «Выйти»

---

### **Навигация**
- 5 вкладок: Главная, Чаты, ИИ-чат (центральная, выделенная), Профсоюз, Профиль

---

### **Иконка приложения**
- Тёмный фон (#09090b → #1e3a5f градиент), белый щит с синей обводкой, внутри стилизованные силуэты людей — символ единства и профсоюза
